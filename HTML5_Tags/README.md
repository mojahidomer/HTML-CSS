# 📘 HTML5 Tags (Complete Guide + Interview Prep)

---

# 🧾 1. What is HTML5?

👉 HTML5 is the latest version of HTML used to structure modern web applications.
It introduced **semantic tags, multimedia support, and better APIs**.

---

# 🧩 2. Semantic HTML5 Tags (Very Important)

👉 These tags give **meaning to the content** (important for SEO + accessibility)

---

## 🔹 Layout / Structure Tags

```html id="x2a8pl"
<header>Header Section</header>
<nav>Navigation Links</nav>
<main>Main Content</main>
<section>Section of Content</section>
<article>Independent Content</article>
<aside>Sidebar</aside>
<footer>Footer Section</footer>
```

---

## 🔹 Explanation

| Tag         | Purpose                          |
| ----------- | -------------------------------- |
| `<header>`  | Top section (logo, title)        |
| `<nav>`     | Navigation links                 |
| `<main>`    | Main content                     |
| `<section>` | Group of related content         |
| `<article>` | Independent content (blog, post) |
| `<aside>`   | Sidebar content                  |
| `<footer>`  | Bottom section                   |

# 💡 8. Best Practices

✅ Use semantic tags
✅ Avoid `<div>` everywhere
✅ Use proper structure (`header → main → footer`)
✅ Improve accessibility

# 🔥 9. Interview Questions & Answers

---

## ✅ Beginner

### 1. What is HTML5?

👉 Latest version of HTML with new features like semantic tags and multimedia.

---

### 2. What are semantic tags?

👉 Tags that describe meaning of content (e.g., `<header>`, `<article>`).

---

### 3. What is `<section>` vs `<article>`?

👉 `<section>` → group of content
👉 `<article>` → independent content

---

## ⚡ Intermediate

### 4. What is `<canvas>`?

👉 Used for drawing graphics using JavaScript.

---

### 5. Difference between `<canvas>` and `<svg>`?

👉 canvas → pixel-based
👉 svg → vector-based

---

### 6. What is `<figure>`?

👉 Groups media with caption.

---

## 🚀 Advanced

### 7. Why semantic HTML is important?

👉 Improves SEO + accessibility + maintainability.

---

### 8. What is `<progress>` vs `<meter>`?

👉 progress → task completion
👉 meter → measurement (range)

---

### 9. What are HTML5 APIs?

👉

* Geolocation
* LocalStorage
* Drag & Drop

---

### 10. How HTML5 improves performance?

👉 Less plugins, native multimedia, better structure.

---
